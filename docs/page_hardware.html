The Hardware

The SN76489 uses 8 digital output lines from the Arduino MCU and an additional digital output to set toggle loading the data into the IC.

 D5 ->  1 +-----+ 16 <- Vcc
 D6 ->  2 | S N | 15 <- D4
 D7 ->  3 |  7  | 14 <- CLK
RDY ->  4 |  6  | 13 <- D3
/WE ->  5 |  4  | 12 <- D2
/CE ->  6 |  8  | 11 <- D1
AUD ->  7 |  9  | 10 <- D0
GND ->  8 +-----+ 9  <- N/C
Signal Description
D0-D7 Command byte inputs
/WE Active low Write Enable (latches data)
VCC 5V
GND Ground
AUD Audio output (headphone jack)
CLK 4MHz clock signal*
/CE Active low Chip Enable (connect to GND or MCU output if more than one IC shares D0-D7)
RDY Ready signal (unused).

NOTE*: The Arduino MCU may have a timer capable of generating the 4MHz clock. This is a hardware dependent function implemented in the startClock() method. The output is restricted to a processor specific pin which can be connected to the IC CLK input.
